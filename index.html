Personal Portfolio & Sandbox

Objective

Hello, my name is Chase Lyall, and I am a business and systems integration consultant for Accenture Federal Services. I am especially interested in how large, crowd-sourced datasets can improve politics and am therefore working to better understand all of the technologies that support large networks and data. My other interests include travelling, skiing, volunteering for the homeless, and hiking/camping.

Website

Just like life, this website will always be a work in progress. Right now I am utilizing it mostly as a learning tool and sandbox. This website is built used to be built using the following web technologies: Django (Python web app framework), Postgres (database), Twitter Bootstrap (frontend mobile-friendly formatting), Amazon Web Services (backend infrastructure), Docker, Gunicorn, Supervisor, Nginx, JQuery, Fontawesome, HTML5 Boilerplate, and several others. Since AWS is prohibitively expensive after the first free year I have switched to a static website using Github pages.

What's to Come

While I have a background in managing large Oracle datasets (and bioengineering), I am hoping to utilize this website to better understand web technologies, their uses, and pitfalls. I believe in lifelong learning, and this website is just one small step along that process. In the future I hope to create a flash card based web app to improve learning retention.
